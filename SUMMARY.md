# AI-Gateway competitive feature-mining ‚Äî Summary

**What this is:** a survey of the GitHub `ai-gateway` topic (201 repos updated since 2026-06-07), to find features/functionality/polish that **Hadrian** could learn from. 200 repos were shallow-cloned inside a `runsc`/gVisor-sandboxed container; **no repo code was executed or built**. They were then read (read-only) and analyzed against Hadrian's existing feature baseline. All findings below are code-confirmed (file paths cited), not README marketing.

**How to read this:**
- **Part 1 ‚Äî Worthwhile repos** (ranked): where to spend your time.
- **Part 2 ‚Äî Prioritized learnings** (10 themes): the concrete, additive ideas, with repo + file citations.
- **Part 3 ‚Äî Per-repo appendix**: full detail for every substantial repo so nothing is lost.
- **Part 4 ‚Äî Malware / junk**: what to avoid in this topic.

> Baseline assumed already in Hadrian (not re-credited as "learnings"): unified OpenAI API incl. Responses; OpenAI/Anthropic/Bedrock/Vertex/Azure + OpenAI-compatible providers; fallback chains, circuit breakers, health checks; org‚Üíteam‚Üíproject‚Üíuser multi-tenancy with scoped budgets/limits; API-key/OIDC/SSO/SAML/SCIM/IAP auth, CEL RBAC, data residency; guardrails (blocklist, PII, moderation) in blocking/concurrent/post modes; exact + semantic cache (pgvector/Qdrant); RAG/Vector Stores API; micro-cent cost tracking + forecasting + atomic budget reservation; Prometheus/OTLP/structured logging; multi-model chat UI (15 modes), web search, frontend tools, MCP; server-side shell tool in containers, server-side MCP, Skills; Studio (image/TTS/transcription/translation); external secrets managers; single binary + Docker + Helm, SQLite + Postgres + Redis + S3. Stack: Rust/Axum + React/TS.

## Part 2 ‚Äî Prioritized learnings (by theme)

### Theme 1 ‚Äî Intelligent / quality-aware routing  ‚òÖ biggest gap (recurred in ~20 repos)
Hadrian routing = static fallback chains + circuit breakers. Peers go far beyond:

- **LLM-classifier intent routing** ‚Äî route to a model/skill/agent declared only as `name: + description:`, decided by a cheap dedicated model; with **session-sticky route caching** and **live-metric re-ranking** (Prometheus p50 + live pricing). `katanemo/plano` (`brightstaff/src/router/orchestrator_model_v1.rs`, `router/orchestrator.rs`, `router/model_metrics.rs`), `litellm/router_strategy/auto_router/`.
- **Adaptive Thompson-sampling bandit** learning per-(request-type, model) quality from **post-call signal detection** (regex + tool-call success/loop/stagnation/satisfaction ‚Üí Œ±/Œ≤ credit), batched to Postgres. `litellm/router_strategy/adaptive_router/` (`signals.py`).
- **Signal ‚Üí boolean rule-tree ‚Üí selection IR** ‚Äî promote existing detectors (PII/jailbreak/moderation) from *gates* to *routing inputs*; ~19 cheap signals (domain, keyword, embedding, complexity, structure e.g. question-count/step-density, conversation/agentic-loop, modality, language, reask) combine in an `AND/OR/NOT` rule tree ‚Üí decision ‚Üí pluggable selector. `vllm-project/semantic-router` (`src/semantic-router/pkg/dsl/`, `pkg/decision/engine.go`, `pkg/selection/`).
- **Entropy-gated reasoning** ‚Äî enable thinking/effort only when the classifier's category softmax is uncertain (Shannon entropy), mapped to each provider's knob. `semantic-router` (`pkg/utils/entropy/entropy.go`, `reasoning_families` adapter).
- **Complexity-tier routing (zero-LLM, sub-ms)** ‚Äî rule scorer across weighted dimensions ‚Üí SIMPLE/MEDIUM/COMPLEX/REASONING ‚Üí cheap-vs-expensive. `litellm/router_strategy/complexity_router/`; `pealmeida/gateswarm-router` (6 tiers + golden-set eval harness flagging over-routing bias); `log1p(word_count)` near-free pre-classifier.
- **Embedding‚Üícluster‚Üíbenchmark-quality lookup** ‚Äî pick the cheapest model good enough *for this prompt's cluster* from a precomputed in-memory quality table (local INT8 ONNX encoder, K=16 centroids, RouterArena/SWE-bench/etc.-derived `quality_means.json`). Turn-type heuristics (probe/title-gen/compaction/sub-agent) hard-pin cheap models and skip the scorer. `workweave/router` (`internal/router/cluster/scorer.go`).
- **Proactive quota/rate-limit-aware selection** ‚Äî skip a provider *before* dispatch if it's over its token budget or unhealthy (not after a 429). APISIX (`ai-proxy-multi.lua` `fallback_strategy="instance_health_and_rate_limiting"`), Envoy AI Gateway (quota-mode routing, PT-vs-on-demand spillover), `litellm-rs` (`LeastBusy`/`RateLimitAware` in `src/core/router/strategy_impl.rs`).
- **Continuous multi-horizon EWMA health ‚Üí soft weighted score ‚Üí weighted-random** load-spreading; **traffic-scaled breakers** (open only above a volume floor, avoid low-traffic false trips); **429 treated as health-neutral**; **cache-aware sticky routing** (prefix-hash boost to provider with warm cache). `AI-Stats` (`apps/api/src/pipeline/execute/routing.ts`, `health.ts`).
- **Ensemble weighted-scoring router with abstention** ‚Äî every signal (cost/latency/health/budget/context-fit/capability) scored 0‚Äì1, non-discriminating policies abstain; optional LLM-as-router with operator `routing_guidance` + live budget; **conversation-sticky routing memory**. `Inebrio/Routerly` (`packages/service/src/routing/router.ts`).
- **Typed per-error-class fallback** (context-window error ‚Üí bigger model; content-policy ‚Üí different provider). `litellm-rs/src/core/router/fallback.rs`.
- **Mid-stream / first-byte failover** ‚Äî before first byte, reroute; after, emit a mid-stream error in the client's own SSE/Gemini/Bedrock framing. **Context-length failover** to a larger-context member without breaker penalty. `MattJackson/busbarAI` (`src/forward.rs`).
- **Explainable `RouteDecision`** ‚Äî `{selected, fallbacks, reason[], rejected[{id,reason}], scores[{id,score,factors{}}]}` surfaced as header + trace + a non-executing `route-preview` endpoint; plus **replay a past request's routing under current config and diff it**. `umutkeltek/switchback` (`crates/sb-core/src/routing.rs`, `sb-server/src/handlers/meta.rs`). Also persisted per-request `routingMetadata` in `theopenco/llmgateway` and routing audit logs in `bifrost`.
- **`willingness_to_pay` single cost/quality dial** + per-request `models` pool + `max_cost`. `tryAGI/Martian` (`openapi.yaml`). Surface what the router *did* (`routed_to/estimated_cost`) in the response.
- **Adaptive metric-driven router** (TTFT + error + inflight + circuit-breaker penalty + 10% exploration). `adrianliechti/wingman` (`pkg/router/adaptive/router.go`).
- **Semantic load-balancing** ‚Äî route by embedding cosine-similarity to specialized upstreams (coding vs translation model). `NeuralTrust/TrustGate` (`loadbalancer/strategies/semantic.go`).
- **Adaptive cost-tier boundaries** computed from p25/p75 of the *actual configured* provider prices. `Das-rebel/a3m-router` (memory layer is unwired; benchmark claim self-fabricated ‚Äî caveat).
- **Routing strategy library to mirror**: `cascade` (cheap‚Üíexpensive graded on confidence), `race` (first-2xx-wins), `shadow` (mirror % for A/B). `soapbucket/sbproxy`. `litellm`: `lowest_cost`/`lowest_latency`/`lowest_tpm_rpm_v2`/`least_busy`/`tag-based`.

### Theme 2 ‚Äî MCP: become an aggregator, not just a consumer (recurred in ~8 repos)
- **MCP federation** ‚Äî front N upstream MCP servers behind one endpoint, namespace `server_tool`, multiplex JSON-RPC streams, with **collision handling** (mcp-mux uses underscore prefix because Cursor only allows `[a-z0-9_-]`; first-come no-stealing). `agentgateway` (`mcp/router.rs`, `mcp/mergestream.rs`), `mcpmux/mcp-mux` (`prefix_cache.rs`), `bifrost`, `protoflux`, `ferrum-edge`, Envoy (`mcp_route.go`).
- **Per-tool RBAC via CEL/JWT scope** + **MCP guardrails** inspecting tool *args and results* in Request/Response/Full phases with wildcard precedence. `agentgateway` (`mcp/rbac.rs`, `mcp/guardrails/`), Envoy, Casdoor (`mcpself/permission.go`).
- **No-code MCP synthesis** ‚Äî generate MCP tools from an OpenAPI spec, a REST endpoint w/ request/response templates, or a DB DSN (auto `query`/`list_tables`/`describe_table`). `higress` (`pkg/mcp/server/rest_server.go`, gorm), `agentgateway` (`mcp/upstream/openapi/`), `reshapr` (OpenAPI/GraphQL/protobuf importers), `protoflux`.
- **MCP tool-drift / "rug-pull" defense** ‚Äî hash name+description+input_schema, alert/hide on silent change (defends prompt-injection-via-tool-description). `OximyHQ` (`gateway-mcp/src/hash.rs`), `elida`, `ferrum-edge`, Oximy.
- **"Code Mode"** ‚Äî let the model write a script calling tools server-side, return only the final result, to kill per-tool-call round-trips + catalog token cost; tools presented as a virtual FS of generated `.pyi` stubs. `bifrost` (`core/mcp/codemode.go`, Starlark), `voidllm` (QuickJS WASM, claims 30‚Äì80% token cut).
- **Per-client capability scoping ("Feature Sets")** ‚Äî composable permission bundles; `tools/list` filtered per grant. `mcpmux` (`feature_set.rs`).
- **MCP server registry + per-principal token broker** ‚Äî upstream MCP servers as first-class resources w/ synced tool lists; mint short-lived scoped tokens so agents reach servers under the gateway's auth, not raw creds. `casdoor`, `bifrost` (`core/mcp/credstore/`, `per_user_oauth`).
- **Inbound MCP via OAuth 2.1 + RFC 7591 Dynamic Client Registration**; full RFC 9728 protected-resource-metadata flow. `mcpmux` (`oauth/dcr.rs`), Envoy, `go-authgate/kong-mcp-oauth2`.
- **Aggregating MCP gateway extras**: BM25 tool search + `defer_loading` to keep big catalogs out of the default list; PII masking of outbound tool args. `protoflux` (`crates/protoflux-core/src/mcp/`).

### Theme 3 ‚Äî A2A (Agent-to-Agent) protocol ‚Äî total gap
- **Agent-card discovery + URL rewriting** for transparent A2A proxying; classify JSON-RPC A2A by method for policy/telemetry. `agentgateway` (`a2a/mod.rs`), `ferrum-edge` (`a2a_gateway.rs`, multi-binding JSON-RPC/REST/gRPC).
- **Agent-registry-as-routable-upstream** with health checks, cost calc, fallback, billed through the same pipeline. `litellm` (`litellm/a2a_protocol/`, `proxy/a2a/`), `litellm-rs` (`src/core/a2a/`).
- **A2A governance** ‚Äî chain-depth cap (429), cycle detection (409), callee allowlist. `sbproxy` (`a2a.rs`).
- **ACP (Agent Client Protocol)** ‚Äî host any coding agent via reverse-RPC JSON-RPC over its stdio; agent calls back for `fs/read`, `fs/write`, `terminal/*`, `session/request_permission`, `elicitation/create`, all path-jailed ‚Äî the host (gateway) owns fs/terminal/permission and brokers every op. `langgenius/mosoo-agent-driver` (`src/runtimes/acp/acp-json.ts`).

### Theme 4 ‚Äî Billing / quota / pricing sophistication
- **Expression-based tiered billing DSL** ‚òÖ ‚Äî one auditable, versioned `expr-lang` formula per model: context-length tiers (`len`), time-of-day (`hour(tz)`/`weekday(tz)`), header-conditional surcharges (after `|||`, e.g. `when(header("anthropic-beta") has "fast-mode")*6`), per-token-category normalization via **AST introspection** (auto-subtracts cache/image/audio sub-tokens only if referenced). Frozen at pre-consume, re-run at settle, version-tagged, UI breakdown. `QuantumNous/new-api` (`pkg/billingexpr/`, design `expr.md`; `service/tiered_settle.go`).
- **CEL token-cost expression + per-model quota buckets + shadow/dry-run mode** ‚Äî `costExpression: "input_tokens + cached_input_tokens*0.1 + output_tokens*6"`, `PerModelQuotas`, multi-bucket burn-down, sliding windows, `ShadowMode` (full check/telemetry, no enforcement). `envoyproxy/ai-gateway` (`api/v1alpha1/quota_policy.go`, `internal/llmcostcel/cel.go`). Also APISIX `limit_strategy=expression`.
- **Prepaid token-credit ledger** with admin grant/refund/query endpoints (`POST /quota/delta`), Redis balance decremented by `total_tokens`, refund-on-error. `higress` (`ai-quota/main.go`).
- **Prepaid reserve/settle credit ledger + transactional billing outbox** ‚Äî pre-debit estimate as a hold, settle vs actuals (supplemental/refund), FIFO segments, sharded hot store + Redis Lua atomic ordered debit; usage rows idempotent (`ON CONFLICT DO NOTHING`) drained by batch worker (no lost records). `neogate-io/NeoGate` (`backend/src/billing/`).
- **Reservations-table-derived "reserved"** (= `SUM(open rows)`, never drifts; crash recovery = 60s TTL sweeper; `BEGIN IMMEDIATE`/`FOR UPDATE` serializes) ‚Äî cleaner than a mutable counter. `OximyHQ` (`docs/2026-06-11-persistence-and-durable-spend-design.md`).
- **One idempotency key across all retries + failover targets** to prevent double-billing when a "failed" retry actually succeeded upstream (per-transport `supports_idempotency`). `OximyHQ` (`gateway-route/src/router.rs`).
- **Cryptographic single-use cost-quote tokens** (EdDSA JWS binding route+price+rail+nonce) + a `BillingRail` trait abstraction. `sbproxy` (`quote_token.rs`).
- **Tool-call surcharges** as first-class billing (web/file/image built-in tools billed per call). `new-api` (`service/tool_billing.go`).
- **Violation-fee billing** (charge a penalty on CSAM/policy upstream errors). `new-api` (`service/violation_fee.go`).
- **Upstream pricing auto-sync with confidence scoring** (pull from another new-api / OpenRouter / models.dev, ETag-cached, flag suspicious ratios, diff before apply). `new-api` (`controller/ratio_sync.go`). DB-backed distributed lock so one node fetches pricing ‚Äî `bifrost` (`configstore/dlock.go`).
- **Pluggable funding sources** (wallet OR subscription, `wallet_first`/`subscription_first`, reserve-then-settle across both; trust-quota fast path skips pre-consume). `new-api` (`service/billing_session.go`).
- **Per-request cost ceilings + efficiency floor** (`max_cost`, `max_cost_per_million_tokens`). `Martian`.
- **Float cost-budget rate limit** ($/day) alongside request/token limits. `wso2/agent-manager`. **Per-model weighted request budgeting** (`request_limit_multiplier` so cheap models cost <1 slot). `TU-Wien/aqueduct`.
- ‚öÝÔ∏è **Likely correctness bugs to check in Hadrian today**:
  - **Tiered/context-threshold pricing** ‚Äî long-context models (Gemini, some GPT) bill higher past a context threshold; flat-rate tables under-bill. `basellm/llm-metadata`, `new-api`, `litellm-rs` (`input_cost_per_token_above_272k_tokens`).
  - **Semantic-cache + redaction cross-contamination** ‚Äî partition the semantic cache by a privacy-signature (`EMAIL:1|IP:1|...`) so redacted/unredacted results never collide. `vcal-project/ai-firewall` (`src/guards/privacy.rs`).
  - **Anthropic `cache_control` breakpoint preservation** ‚Äî gateways that strip it silently kill the prompt-cache discount. `modelmeld` (`adapters/anthropic_adapter.py`).
  - **Semantic cache must include the `stream` flag** in scope (a non-stream entry must never replay to a `stream:true` caller). `ferrum-edge`.

### Theme 5 ‚Äî Model catalog / metadata (gap)
- **Adopt the `basellm/llm-metadata` schema wholesale** (`src/types/index.ts`, real data in `dist/api/models/`): capability flags (`attachment, reasoning, tool_call, temperature, structured_output, interleaved, open_weights`); `limit.context/output`; per-direction `modalities.input/output` (`text|image|audio|video|pdf`); `family`, `knowledge` (cutoff), `release_date`; `reasoning_options:[{type:"budget_tokens",min,max}]`. **Pricing**: `input/output/cache_read/cache_write/reasoning` + `currency`; **tiered** `tiers:[{tier:{type:"context",size:200000},...}]`; **per-modality** (`text_input/vision_input/audio_input/embedding_*`, `per_second/per_image/per_10k_chars`); dynamic segmented keys (`input_32k_128k`, `thinking_output_256k_1m`).
- **Override-layer build with hash-verified manifest** ‚Äî merge upstream source + per-provider/model overrides + policy, emit `manifest.json` (`sourceHash/overridesHash/policyHash`, counts, `generatedAt`); ship both a fat `all.json` and a slim flag-only `index.json` for the UI picker; i18n names/descriptions. `basellm/llm-metadata`.
- **models.dev as a hot-reloaded data-driven catalog** (`ArcSwap` swap; new models work day-of, no redeploy; remote URL override + embedded fallback + never-block-startup). `OximyHQ` (`gateway-cache/src/hot_registry.rs`), `nyro` (bundled snapshot), `ferro-labs/ai-gateway` (`FERRO_MODEL_CATALOG_URL`), `new-api`.
- **Machine-readable provider √ó endpoint capability matrix** drives routing/validation. `litellm` (`provider_endpoints_support.json`, 91KB).
- **Authored catalog reshaped per calling agent** ‚Äî `/v1/models` metadata owner-controlled and re-shaped per client (Codex schema vs opencode `limit:{}` block) for drop-in compat. `1flowbase`, `nyro`.
- **"Missing models" gap detection** auto-creating catalog entries from models.dev/basellm. `new-api` (`model/missing_models.go`).

### Theme 6 ‚Äî Prompt management (whole category Hadrian lacks)

- **Prompt registry with versions/rollback + composable partials + render-without-execute endpoint** (`/prompts/{id}/render` returns the substituted prompt without calling a model) + `/collections`. `tryAGI/Portkey` (`PromptPartial.g.cs`).
- **Prompts as versioned, observable, billing-reconciled assets** ‚Äî "Actions" (model+prompt+typed vars w/ `ActiveVersionId` for instant A/B/rollback), "Templates" (SPLITTER‚ÜíBRANCH‚ÜíMERGE workflows). `tripplemay/aigcgateway` (`src/lib/mcp/`, `src/lib/billing-audit/`).
